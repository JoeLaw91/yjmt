## Environment to prepare
- Ensure that xampp is running
- Ensure that database table (e.g.: yjmt) is created
- Close the repository to the directory you wanted
- Update the database credentials in .env file accordingly

## Commands to Run While Initializing
- cp .env.example .env
- php artisan key:gen
- composer install
- php artisan migrate
- php artisan db:seed
- php artisan storage:link
